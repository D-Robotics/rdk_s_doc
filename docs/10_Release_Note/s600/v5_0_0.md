# RDKS600_LNX_SDK_V5.0.0_BETA

**发布日期：** 2025-12-19

## 1. 主要信息

**RDK S600 V5.0.0_BETA** 发布内容包括：

- **Linux Kernel：** 6.1.158-rt58
- **Bootloader：** U-Boot 2022.04
- **Root File System：** Based on Ubuntu 24.04 (noble)，基于 Multistrap 构建
- **支持的 Host 系统：**
  - 编译环境：Ubuntu 22.04 (x64)
  - 烧录环境：Ubuntu 22.04 (x64)
- **交叉编译工具链：** arm-gnu-toolchain-13.2.Rel1 (aarch64)
- **其他组件：** D-Robotics 驱动、必要固件、工具链及实用工具

## 2. 软硬件功能清单
### 2.1 功能列表

| 接口     | 描述                                                                                                                                            |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Display  | HDMI显示支持1080P/2K 60FPS                                                                                                                      |
| Storage  | 支持NVME硬盘，PCIE4.0x2                                                                                                                         |
| Audio    | 能够支持USB声卡输入输出，蓝牙声卡输出                                                                                                           |
| Ethernet | 主域：<br/>2个千兆网口<br/>PTP协议<br/>EtherCAT单主站（IGH）                                                                                    |
| Wireless | 支持Wi-Fi 6 Station模式<br/>支持蓝牙                                                                                                            |
| USB      | 支持6路USB 3.0接口                                                                                                                              |
| Camera   | 6路GMSL环视，同步触发<br/>2路MIPI相机同步<br/>支持2路GMSL/USB 深度相机接入                                                                      |
| 闪连接口 | 支持烧录<br/>支持串口调试<br/>默认支持ADB调试，可手动切换为RNDIS                                                                                |
| 风扇     | 支持温控策略自动调速                                                                                                                            |
| 时间同步 | 支持时间同步<br/>支持ACore MCU时间同步<br/>支持LPWM同步触发                                                                                     |
| MCU      | ACore RemoteProc支持Start/Stop/Log获取<br/>支持4路CANFD透传，1000Hz通信频率，有示例程序（握手1M+通讯8M）<br/>支持ADC/PWM/I2C/SPI/UART等外设传输 |
| 桌面应用 | 基于Ubuntu24.04<br/>支持 Nomachine                                                                                                              |
| 应用示例 | π0-硬件在环<br/>多模态交互示例（7B）                                                                                                            |

### 2.2 已知问题
- 内核配置为64K页表，Ubuntu桌面应用有兼容性问题；
- 万兆网口正在调试中；
- ACore CAN接口正在调试中；
- ACore MCU不支持Alive/CpuLoad等数据统计；
- 硬件状况请参考[RDK S600 早期样机局限性列表](https://horizonrobotics.feishu.cn/wiki/LyjewVlbZiUOdSkBGdocgdxhngd?from=from_copylink)。
