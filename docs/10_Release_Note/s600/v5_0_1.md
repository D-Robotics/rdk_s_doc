# RDKS600_LNX_SDK_V5.0.1_BETA

**发布日期：** 2026-03-11

## 1. 主要信息

**RDK S600 V5.0.1_BETA** 发布内容包括：

- **Linux Kernel：** 6.1.158-rt58
- **Bootloader：** U-Boot 2022.04
- **Root File System：** Based on Ubuntu 24.04 (noble)，基于 Multistrap 构建
- **支持的 Host 系统：**
  - 编译环境：Ubuntu 22.04 (x64)
  - 烧录环境：Ubuntu 22.04 (x64)
- **交叉编译工具链：** arm-gnu-toolchain-13.2.Rel1 (aarch64)
- **其他组件：** D-Robotics 驱动、必要固件、工具链及实用工具

## 2. 本次发布包含的主要功能更新/变更
- 支持系统镜像的备份和再次烧录。

### 2.1 硬件支持
- 支持 V0P2版本硬件。

### 2.2 ACore
#### 2.2.1 Thermal
- 调整 thermal 策略，芯片结温45度下风扇开始，芯片结温115度下关机；
- 调整 PVT sensor 精度到千分之一摄氏度或毫伏。

#### 2.2.2 Camera
- 支持 ORBBEC Gemini 335LG 深度相机；
- V4L2支持使用 RX1 RX3 CPHY 模式。

#### 2.2.3 LSIO
- 调整 UART 总数至8个，支持 ttyS4~7；
- 支持 ICM-42688 IMU 驱动(I2C)。

#### 2.2.4 VPU
- 支持 V4L2框架；

### 2.3 OTA
- 新增 hooks 脚本机制，用户可在分区升级前后新增自定义脚本；
- 新增 zstd 的升级包压缩格式，加快压缩解压速度，依赖 zstandard 包。

### 2.4 MCU
- 提供硬件定时器的 sample；
- 支持 BMI088  IMU 传感器驱动；
- 支持 uart 波特率230400/460800；
- 支持 mcu1 release 版本中的 console 命令；
- 支持 SPI4数据传输测试；
- 支持 ADC 连续转换接口和 private 接口；
- 支持 MCU1的正常启停睡眠功能。

### 2.5 Xburn 烧录工具
- 更新 Xburn 烧录工具，版本 1.1.9，替代 D-navigation。

### 2.6 security
- 支持使用硬件加解密引擎；
- 支持不依赖于 optee 源码进行 TA 的开发。

## 3. 本次发布包含的重点问题修复
### 3.1 MCU
#### 3.1.1 外设
- 修复睡眠按钮功能的异常；
- 修复整机下电不彻底的异常问题；
- 修复 I2C 同步/异步接收数据异常的问题；
- 新增 eth pps 中断注册功能，支持配置 eth pps 作为 timeync_sample 的触发源；
- 修复 pps 触发模式配置错误导致中断风暴 mcu 异常挂掉的问题；
- 修复 can 轮询方式，两块板子进行收发数据，数据错乱问题；
- 修复 SPI4 CS1不能正常使用问题；
- 修复内部 RTC 无法读取时间问题；
- 修复 MCU1核间通信异常，DMA 中断未响应问题。

#### 3.1.2 系统
- 修复 MCU1 Core0/1 sleep/stop 异常的问题；
- 在 Flash 处理任务中加入 vTaskDelay，避免长时间占用 cpu 资源影响其他任务的运行；
- 更改 MS_TO_TICK 延时接口，规避原有接口的除0风险；
- 调整 mcu1的内存布局适配现有功能，解决与 IPC 已分配的内存冲突。

## 4. 已知问题
### 4.1 RDK Linux
- 部分 Ubuntu 应用在 GPU 加速下因页大小不匹配崩溃；
- 在不插入 HDMI 线时，RDK S600的桌面系统(GDM3)会默认回退为 X11渲染，S600目前不支持 X11的 GPU 加速，回退后会使用 CPU 渲染，导致 CPU 占用升高；
- 万兆网口启动后进行固件下载有性能问题，需要等待固件下载成功后才能达到万兆性能；

### 4.2 MCU SDK
- 目前 MCU 系统有部分稳定性问题；
- 目前不支持锁步解锁。
