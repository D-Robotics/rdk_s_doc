# hrut_socuid

URL: https://developer.d-robotics.cc/rdk_s_doc/en/Appendix/rdk-command-manual/cmd_hrut_socuid

The **hrut_socuid** command prints the UID (Unique Identifier) of the current SoC chip.

## Syntax

```text
hrut_socuid
```

## Common Usage

```shell
sunrise@ubuntu:~$ hrut_socuid
060c0b0d3090694108255c4c00001079
```
