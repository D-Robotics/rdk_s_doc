# 5 TogetheROS.Bot  
Introduction to TogetheROS.Bot  
TogetheROS.Bot is a robot operating system launched by D-Robotics for robot manufacturers and ecosystem developers. It aims to unlock the intelligent potential of robot scenarios, helping ecosystem developers and commercial customers to carry out robot development efficiently and conveniently, and to create competitive intelligent robot products.

TogetheROS.Bot supports operation on the RDK platform and also provides a simulator version for operation on X86 platforms. The RDK platform includes all the features shown in the figure below, while the X86 platform supports experiencing some functions through image replay, improving the efficiency of algorithm development and validation, and enabling quick migration to the RDK platform.

The source code of TogetheROS.Bot is hosted on GitHub under the [D-Robotics organization](https://github.com/D-Robotics).
